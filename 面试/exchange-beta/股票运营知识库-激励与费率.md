## 一、四种合规激励形式详解

### 1. 手续费折扣券

- **逻辑**：平台补贴佣金差额，而非直接给钱。传统券商（富途、老虎）都在做，合规边界清晰。
- **设计**：阶梯式 — 首笔送 7 折、月交易 5 笔送 5 折、月交易 20 笔送免佣。
- **合规安全**：降低已有交易行为的摩擦成本，非诱导交易。

### 2. Level2 行情权益

**L1 vs L2 本质区别**

L1 只告诉你「现在什么价」，L2 告诉你「市场在想什么」。

| 信息 | L1 实时行情 | L2 深度行情 |
|------|-----------|-----------|
| 最新成交价 | ✅ | ✅ |
| 涨跌幅/成交量 | ✅ | ✅ |
| 买一~买十挂单 | ❌ | ✅ |
| 卖一~卖十挂单 | ❌ | ✅ |
| 每笔挂单的量 | ❌ | ✅ |
| 逐笔成交明细 | ❌ | ✅ |

**举例**：AAPL 报价 $185.00
- L1 用户：$185.00，涨 1.2%，只能凭感觉买。
- L2 用户：卖一 $185.01 挂 500 股，卖二 $185.05 挂 2 万股，买一 $184.98 挂 8 万股 → $184.98 有强支撑，上方卖压很轻，大概率突破 $185。这就是信息差。

**传统券商价格参考**：L2 深度 $5-20/月，L2+ 全深度 $30-100/月。

**对 ExchangeBeta 的价值**：
- xStocks 用户最需要 L2，订单簿是预判涨跌的核心工具
- 30 天免费 → 到期后「月交易满 5 笔继续免费」→ 制造切换成本
- 边际成本极低，交易所基础设施固定成本

### 3. 碎股赠送

- **实现**：用户完成股票账户开通 → 随机获赠 $1-$10 等值 NVDA/TSLA/AAPL 热门标的碎股盲盒。
- **比 USDT 好在哪**：送碎股用户要卖出才能变现 → 过程打开交易页面、看到持仓、看到涨跌 → 第一次交易体验入口。
- **防羊毛**：碎股持仓锁定 7-14 天才能卖出。正常用户不介意。
- **心理效应**：endowment effect — 用户收到 NVDA 碎股后开始关注 NVDA 股价。

### 4. IPO 申购加权

- **逻辑**：完成首笔股票交易 → IPO 申购权重 +20%。不给钱，给稀缺资源的优先分配权，边际成本几乎为零。
- **设计**：新用户开户 → 1x；首笔交易 → 1.5x；月交易满 5 笔 → 2x。
- **ExchangeBeta 独特优势**：ExchangeEpsilon 没有 IPO 申购，此激励无法复制。

### 四者组合链路

```
注册 → KYC → 开户 → [碎股盲盒] 触发首次关注
                ↓
            入金 → [手续费7折券] 降低交易摩擦
                ↓
            首笔交易 → [L2行情1个月] 提升交易体验
                ↓
            月交易5笔 → [IPO申购2x权重] 长期留存
```

## 二、各平台手续费对比

### 传统券商

| 券商 | 美股佣金 | 平台费 | 特点 |
|------|---------|--------|------|
| 盈透 IBKR | $0.005/股，最低 $1 | 无 | 机构级，阶梯定价 |
| 富途 moomoo | $0.0049/股，最低 $1.99 | $0.005/股 SEC费 | 散户友好 |
| 老虎 | $0.005/股，最低 $1.99 | 同上 | 和富途对标 |
| 嘉信 Schwab | **$0** | 无 | PFOF 模式，美国零佣金先驱 |

> 零佣金 ≠ 不赚钱。传统券商靠订单流返利（PFOF）— 把用户订单卖给 Citadel 等做市商赚差价。

### 加密交易所股票产品

| 平台 | 股票产品 | 费率 | 定位 |
|------|---------|------|------|
| **ExchangeBeta** | 真实股票 | 走 TradingAPI 通道，参照券商标准 | 非核心收入源 |
| **ExchangeBeta** | xStocks (衍生品) | 标准合约费率 | 平台自主定价 |
| **ExchangeEpsilon** | 真实股票 | **零佣金**（限时） | 烧钱抢市场，3亿用户规模摊薄 |
| **ExchangeZeta** | Stocks 2.0 (rTokens) | **0.05%** maker/taker，VIP 最低 0% | 四家最低，BGB折扣后近零 |
| **ExchangeDelta** | xStocks | 合约标准费率 | IPO申购不额外收费 |

### 竞争定位

```
高佣金 ←————————————————————————→ 零佣金
 传统券商           Gate真实股票    ExchangeZeta       RetailBroker
(富途$1.99起)      (走TradingAPI)     (0.05%)      ExchangeEpsilon(限时免佣)
```

### RetailBroker

> 0 佣金 + 碎股 $1 起买 + 界面极简。PFOF 模式。2024 年月活 ~2400 万。是加密交易所股票业务的头号对标。

## 三、手续费7折券的真实价值

在 ExchangeBeta 场景下，真实股票走 TradingAPI 通道，平台对佣金话语权有限。「7 折券」本质：

> ExchangeBeta 补贴 30% 佣金差价，平台承担成本，用户感知到优惠。

和 crypto 交易所「手续费返佣」逻辑一致 —— 不改变费率结构，用补贴降低决策摩擦。尤其 crypto 用户习惯了低费率，对传统券商 $1.99 起收的佣金会很敏感，30% 折扣对新用户第一笔交易的心理影响很大。
