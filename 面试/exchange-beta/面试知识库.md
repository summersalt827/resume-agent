## 一、激励形式

### 手续费折扣券
平台补贴佣金差额，非直接给钱。传统券商（富途、老虎）都在做，合规边界清晰。
阶梯设计：首笔 7 折 → 月 5 笔 5 折 → 月 20 笔免佣。

### L2 行情权益

**L1 vs L2**

| 信息 | L1 | L2 |
|------|----|----|
| 最新成交价 | ✅ | ✅ |
| 买一~买十 / 卖一~卖十挂单 | ❌ | ✅ |
| 逐笔成交明细 | ❌ | ✅ |

L1 告诉你「现在什么价」，L2 告诉你「市场在想什么」。传统券商 L2 收费 $5-20/月。

举例：AAPL $185.00 — L1 用户只看到价格，L2 用户看到 $184.98 有 8 万股强支撑 → 预判突破方向。

对 ExchangeBeta 价值：xStocks 交易者核心工具，送 30 天到期后「月交易 5 笔续期」制造切换成本。边际成本极低。

### 碎股赠送
送 $1-$10 等值热门标的碎股盲盒（NVDA/TSLA/AAPL）。比送 USDT 好：用户要卖出才变现，过程中打开交易页、看到持仓 → 首次交易体验入口。持仓锁定 7-14 天防羊毛。心理效应：endowment effect — 收到就开始关注股价。

### IPO 申购加权
首笔交易 → 申购权重 +20%。给稀缺资源优先分配权，边际成本为零。设计：开户 1x → 首交 1.5x → 月 5 笔 2x。ExchangeEpsilon 没有 IPO → 无法复制的差异化锚点。

### 四者组合链路
```
注册 → KYC → 开户 → [碎股盲盒] 触发关注
                ↓
            入金 → [手续费7折券] 降低摩擦
                ↓
            首笔交易 → [L2行情] 提升体验
                ↓
            月5笔 → [IPO申购2x] 长期留存
```

## 三、加密用户 vs 传统股票用户画像差异

| 维度 | 加密原生用户 | 传统股票用户 |
|------|------------|------------|
| 交易时间习惯 | 24/7，随时操作 | 盘前盘后，周一~周五 |
| 费率敏感度 | 极敏感（习惯万分之一级别） | 较敏感但接受 $1.99 起 |
| 决策周期 | 分钟/小时级 | 天/周级 |
| 信息获取 | TG/Discord/X/TikTok | 财报/券商研报/彭博 |
| 风险偏好 | 高，习惯杠杆 | 中低，多数只做多 |
| 对平台信任 | 经历过去中心化叙事，对CEX有戒心 | 对持牌券商有天然信任 |
| 产品认知 | 熟悉合约/杠杆/永续 | 熟悉股票/ETF/股息 |

> 核心洞察：两类用户在同一个平台相遇，运营要做翻译器 — 用 crypto 用户懂的语言讲股票，用股票用户信任的方式做合规。

## 六、富途牛牛会员权益体系

> 面试参考：传统券商会员体系标杆，为 ExchangeBeta 股票业务激励设计提供对标。

### 会员等级门槛

- 等级分 ≥ 888 分晋升为牛牛会员
- V1 → V2 → V3 三级递进（资产越高、交易越活跃，等级越高）

### 核心权益（约 11-15 项）

| 类别 | 权益 | 说明 |
|------|------|------|
| **行情数据** | L2 实时深度行情 | 买卖盘口十档、逐笔成交，传统券商收费 $5-20/月 |
| | Premium 机构级数据 | K线资金流模型、筹码分布、技术面+基本面共振研判 |
| | 美股 TotalView | Nasdaq 全深度订单簿，机构级工具 |

### 会员成长驱动（行为 → 等级分）

| 行为 | 等级分 | 说明 |
|------|--------|------|
| 日均资产 | 主要来源 | 持仓市值 + 现金余额 |

## 七、ExchangeBeta 三层产品差异化打法（Q15 参考答案）

### 核心提案：统一持仓 + 跨产品抵押

这是只有 ExchangeBeta 能做到的事——三层产品（xStocks / Ondo / 真实股票），而 ExchangeDelta 只有 xStocks、ExchangeZeta 只有代币化股票、ExchangeEpsilon 没有衍生品层。

### 具体方案

**统一持仓视图：**
真实股票 AAPL + xStocks AAPL 做多 + Ondo AAPL RWA 代币 → 一个页面展示所有 AAPL 相关持仓的总价值、总收益。

**跨产品抵押：**
- 用户持有 AAPL 真实股票 → 可作为保证金开 xStocks 合约
- 用户持有 Ondo RWA 代币 → 可抵押借 USDT 继续交易

### 竞争优势

1. **降低用户认知成本**：三套产品不再是割裂的，用户清楚知道自己持仓了多少 AAPL
2. **提高资本效率**：不必卖股票才能开合约，持仓资产可复用。比 ExchangeZeta 只支持 15 只代币化股票作保证金范围更广
3. **形成产品飞轮**：真实股票 → 持仓增值 → 释放保证金 → 开 xStocks 合约 → 盈利/亏损 → 回到真实股票调仓。三套产品从「独立」变「联动」，用户粘性和 ARPU 双升

### 落地路径

| 阶段 | 时间 | 内容 |
|------|------|------|
| Phase 1 | 1-2 月 | 统一持仓视图（只读聚合，产品改动最小，运营价值最大） |
| Phase 2 | 3-4 月 | 真实股票 → xStocks 保证金试点（选 5-10 只核心标的，需法务确认） |
| Phase 3 | 5-6 月 | 全产品线打通（Ondo 纳入抵押品 + 跨产品一键转换） |

### 为什么别人难抄

- ExchangeDelta：没有真实股票
- ExchangeZeta：没有衍生品股票层
- ExchangeEpsilon：有真实股票但没有 xStocks 期货层

三层架构是 ExchangeBeta 独有，这个打法基于架构优势而非单点功能。
