## 七、关键结论

1. **KOL/邀请是绝对主力渠道** — 贡献 35-40% 新增, 55% 的最高返佣 + 终身结算使 ExchangeAlpha 在 KOL 圈有强竞争力
2. **空投中心是流量入口之王** — COO 明确称其为「平台流量最高的入口之一」, 但用户质量偏低, 需后续留存策略承接
3. **东南亚 + 东欧 = 半壁江山** — 两个区域贡献 ~52% 新增用户, 是所有增长策略的优先级锚点
4. **付费投放相对克制** — COO 强调「留存率 > 新用户增长」, 投放不激进, 占比估计仅 10-15%, 远低于一线交易所
5. **产品驱动增长 (PLG) 在起步** — RWA 专区、矿币专区、盘前 OTC、AI 工具等差异化产品自带获客能力, 2025 年贡献 10% 新增, 预计 2026 年进一步上升
6. **非洲是最大变量** — 当前占比 10% 但增速最快, 空投 + 任务中心模式在该区域 ROI 极高, 瓶颈在支付通道
